# 🧠 Responsive Quiz App

A responsive and interactive Quiz Application built using **HTML, CSS, and JavaScript**. The app presents multiple-choice questions one at a time, tracks the user's score, and provides instant feedback with a clean and user-friendly interface.

## 🚀 Features

* ✅ Multiple-choice questions
* ✅ One question displayed at a time
* ✅ Next button navigation
* ✅ Submit button on the final question
* ✅ Score calculation and result display
* ✅ Restart quiz functionality
* ✅ Progress counter and progress bar
* ✅ Correct and incorrect answer highlighting
* ✅ Responsive design for mobile and desktop
* ✅ Smooth hover effects and fade-in animation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📱 Responsive Design

The application is optimized for:

* Desktop Devices
* Tablets
* Mobile Devices

## 🎯 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
